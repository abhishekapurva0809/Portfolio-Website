# Abhishek Apurva - Personal Portfolio

Welcome to my personal portfolio repository! This workspace contains the source code for my professional portfolio website, where I showcase my skills, projects, and educational background as a Software Engineer.

## 🚀 About the Project
This project is built to demonstrate my abilities in frontend web development. It features a fully responsive design, modern UI/UX principles, and multiple technology stacks.

### ✨ Features
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Dark/Light Mode**: Integrated theme toggling for better accessibility and user preference.
- **Dynamic Navigation**: Interactive custom sidebar navigation.
- **Sections**: Home, About, Projects, Skills, Education, and Contact sections detailing my journey.

## 🛠️ Technologies Used
This repository includes different versions of the portfolio, demonstrating my learning process across frontend technologies:
- **Vanilla HTML / CSS / JavaScript**: The core static version of the website (located in the root directory).
- **React.js**: A component-based version located in the `portfolio-react` directory.
- **Next.js**: A server-side rendered version located in the `portfolio-nextjs` directory.
